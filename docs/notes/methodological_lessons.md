# Methodological Lessons from Kuramoto Regime Study

## 1. BC-driven perturbations

Perturbations originate from the boundary condition being studied.

Example:

κ → κ + δκ

This avoids arbitrary state perturbations and ties regime detection to structural variation.

---

## 2. Observable resolution (ε)

ε defines the smallest observable change considered meaningful.

Partition condition:

|Π(κᵢ₊₁) − Π(κᵢ)| > ε

This parameter defines the **scope resolution** of the analysis.

---

## 3. BC resolution (Δκ)

Δκ determines the smallest BC perturbation applied.

Therefore Δκ sets the **lower bound of detectable regimes**.

If Δκ is too large:

small regimes cannot be detected.

---

## 4. Proper experimental design

When comparing different ε values:

Δκ must remain constant.

Correct procedure:

1. Choose smallest ε of interest
2. Choose Δκ fine enough for that ε
3. Run one common BC sweep
4. Evaluate multiple ε on the same data

This ensures that differences arise from ε rather than sampling artifacts.

---

## 5. Regime robustness

Regimes that persist across multiple ε values can be interpreted as **scope-robust structures**.

Structures appearing only at very small ε may represent:

- fine dynamical structure
- or discretization artifacts.

---

## 6. Implication for ARW

Regimes are not purely properties of the system.

They emerge from the interaction of:

- system dynamics
- BC perturbations
- observable resolution.

This supports the ARW view that regime structures are **scope-dependent descriptions of dynamics**.
