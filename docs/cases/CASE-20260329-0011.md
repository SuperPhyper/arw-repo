# Case Motivation: CASE-20260329-0011
## Labyrinth Agent — Designed Modes, Phase 1

**Case ID:** CASE-20260329-0011
**System:** Context-navigation labyrinth agent (prescribed mode library)
**Primary BC class:** Restriction + Coupling
**Pipeline artifacts:** [cases/CASE-20260329-0011/](../../cases/CASE-20260329-0011/)

---

## Why this system?

The labyrinth agent is the ARW framework's first **endogenous cognitive test case**.
In all prior cases (Kuramoto, Doppelpendel, Stuart-Landau), the system's regime
structure is analytically derivable from its equations of motion — the ARW pipeline
validates against known ground truth.

In this case, the agent's regime structure is not given by physics but **constructed
by architectural design**. The agent has a prescribed mode library M = {m₁, ..., mₖ}
and must navigate between modes in response to environmental boundary conditions.
This makes it the first case where the system designer and the ARW analyst are
aligned: the designer prescribes modes; the analyst asks whether those modes produce
a measurable BC-structured partition.

**What this case uniquely tests:**
- Whether ARW partition analysis recovers the prescribed mode structure as a
  behavioral partition (does architectural design translate to measurable regime structure?)
- Whether the zone topology of the labyrinth (R, C, F zones) induces the predicted
  BC-class sub-scopes in the agent's behavior
- Whether the designed regime partition satisfies the same ε-stability criteria as
  the physically-grounded calibration cases

---

## BC Class: Restriction + Coupling

**Primary BC class: Restriction**

The agent's architecture is itself a Restriction-class boundary condition:
the mode library M fixes the set of admissible behavioral strategies.
The agent cannot express behaviors outside M — the mode library restricts
the accessible behavioral state space.

Zone-type constraints (R zones: tight resource constraints) further instantiate
Restriction-class BCs at the environment level.

**Co-driver: Coupling**

Zone C environments enforce Coupling-class constraints: the agent's behavior
must coordinate with local environmental features (coupling to zone-C cues).
The agent's perceptual coupling (visibility radius r=5) is also a Coupling
BC, mediating between environmental structure and behavioral differentiation.

**What is non-trivial about this BC classification:**
The designed mode library M is a Restriction BC *on the architecture*, not just
on the environment. This is a novel BC type in the ARW case portfolio: a BC
that constrains the agent's solution space rather than the physical system's
state space. The labyrinth is the first case where a cognitive architecture
component (the mode library) is itself a formal boundary condition.

---

## What this case adds

| Prior gap | This case's contribution |
|-----------|--------------------------|
| All BC-structured cases are physical dynamical systems | First cognitive/computational system with BC-structured regime partition |
| Regime structure is always analytically derived | Regime structure is architecturally designed — ARW tests whether design = measurable partition |
| No cases with endogenous regime structure | Mode library makes regime structure endogenous (not given by physics) |
| No cognitive architecture BC taxonomy test | First test of whether the BC classes (Restriction, Coupling, Forcing) from `bc_taxonomy_cognitive_modes.md` are empirically recoverable |

**Phase 1 specific contribution:**
Phase 1 uses a single uniform Zone A environment (no zone switching). Its role is
to confirm that a discrete behavioral partition exists at all before introducing
multi-zone complexity. A positive Phase 1 result (stable ε-plateau at N ≥ 1)
is a prerequisite for interpreting all subsequent labyrinth phases.

## Complement experiment

CASE-20260329-0011 (Designed Modes) and CASE-20260330-0012 (Emergent Modes) are
complementary. This case tests whether prescribed architectural modes produce a
measurable ARW partition. CASE-0012 tests whether such a partition arises *without*
prescription. Together they bracket the question of what role architectural design
plays in regime formation.
