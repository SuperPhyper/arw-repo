# ARW for Ecologists

## The Short Version

Ecology already thinks in regimes. Alternative stable states, regime shifts,
tipping points, resilience — these are all regime-theoretic concepts. What
ARW adds is a formal account of *why* a particular description produces
the regime structure you observe, and *when* that description loses validity.

The central ARW claim, in ecological terms: the regime structure you see
depends on which observable you track, at which resolution, under which
perturbation budget. Change those choices and the regime structure changes —
sometimes dramatically.

---

## Vocabulary Translation

| Ecology | ARW | Notes |
|---|---|---|
| State variable (population size, biomass) | Observable Π | Maps system state to measurable quantity |
| Environmental parameter (temperature, nutrient load) | Sweep parameter θ | The axis along which regimes are identified |
| Alternative stable state | Regime | Stable partition element under a scope |
| Regime shift / tipping point | Regime boundary θ* | Transition between partition elements |
| Resilience (Holling) | ε-plateau width w | Wide plateau = robust regime; narrow = fragile |
| Perturbation / disturbance | Δ (admissible perturbations) | What the regime must survive to count as stable |
| Carrying capacity / resource limit | BC class: Restriction (S1) | Resource bound constrains admissible population states |
| Species interactions (competition, predation) | BC class: Coupling (S2) | Cross terms between population state variables |
| Seasonal / climate forcing | BC class: Forcing (S3) | Exogenous time-coupled constraint |
| Population decay / density-dependence | BC class: Dissipation (S4) | Contraction toward carrying capacity |
| Aggregation (community → ecosystem) | BC class: Aggregation (S1 quotient) | Projection from species-level to community-level description |
| Coarse-graining (species → functional groups) | S1 quotient projection | Identifies species as equivalent within functional group |

## What ARW Adds

**1. Resilience is ε-plateau width, not basin depth.**
Holling's resilience is usually operationalized as basin size — how far the
system can be perturbed before shifting regimes. ARW operationalizes it
differently: the ε-plateau width w = log(ε_max/ε_min) measures how wide
the resolution range is over which a regime partition remains stable. A
wide plateau means the regime is robust to changes in observational
resolution, not just to perturbations in state. These are complementary
measures, not substitutes.

**2. The SIR model is the canonical Aggregation-BC case.**
SIR/SEIR compartmentalization is the most explicit Aggregation BC in the
repo: the projection from N individual micro-states to three macro-compartments
(S, I, R) is an S1 quotient projection. The regime partition (sub-threshold /
super-threshold epidemic) is generated by this aggregation, not by some
other structural mechanism. CASE-20260315-0007 is the direct entry point.

**3. Regime shifts have a BC signature.**
Different types of ecological regime shifts have different BC signatures:
- Eutrophication (nutrient → algal bloom): Restriction BC, threshold crossing
- Predator removal → prey explosion: Coupling BC collapses
- Climate-driven shift: Forcing BC modulates existing equilibrium
- Population collapse: Dissipation BC dominates over Coupling

Identifying the BC class of a regime shift predicts its structural
properties: which observable will show the sharpest transition, how
wide the ε-plateau should be, what the transfer score to analogous systems
will be.

**4. Early warning signals are observable insufficiency.**
Critical slowing down, variance increase, and autocorrelation rise near
tipping points are, in ARW terms, signatures of approaching observable
insufficiency: the observable's span relative to ε is shrinking as the
system approaches θ*. ARW's F3 failure mode (no robust ε-plateau) is the
formal version of this phenomenon.

## Where the Tension Lies

**ARW is observer-relative; ecology often assumes observer-independence.**
Ecological regime shifts are usually treated as properties of the system —
the lake eutrophies regardless of who is measuring it. ARW treats regime
structure as scope-dependent: a different choice of observable or resolution
can make the same physical transition invisible or reveal additional
transitions. This is a feature for theoretical ecology but may be
uncomfortable for applied ecology where the regime shift is "real" and
consequential.

**Aggregation is a BC, not a problem to be solved.**
Ecologists often treat the gap between individual-level and population-level
descriptions as a modeling challenge to be overcome. ARW treats it as a
structural feature: the aggregation projection defines the scope, and the
scope determines which regimes are visible. A more fine-grained scope
(individual-based model) is not "better" — it is a different scope with a
different regime structure.

**Lotka-Volterra is Coupling BC, not just a model.**
The Lotka-Volterra equations instantiate S2 (Product ∘ Composition): two
population state variables with cross terms (αxy for predation). The regime
structure (oscillation, coexistence, competitive exclusion) is generated by
the Coupling BC. Adding carrying capacity (logistic terms) adds a Dissipation
BC (S4) as secondary. ARW does not replace Lotka-Volterra analysis — it
classifies it structurally.

## Direct Entry Points

**CASE-20260315-0007 (SIR, Aggregation BC):**

| Ecology concept | SIR/ARW instantiation |
|---|---|
| Population | N individuals with states S/I/R |
| Aggregation | π_SIR: individual states → compartment fractions |
| Control parameter | β (transmission rate) |
| Tipping point | β* = γ_r/S₀ ≈ 0.101 (R₀ = 1 threshold) |
| Sub-threshold regime | No epidemic: I → 0 |
| Super-threshold regime | Epidemic wave: I_peak >> I(0) |
| Resilience | ε-plateau width for I_peak observable |

**CASE-20260315-0005 (Multi-Pendel with damping, Dissipation BC):**
Analog of density-dependent population regulation: coupling (species
interactions) competes with dissipation (density dependence). γ-sweep
is the analog of varying density-dependent mortality rate.

## Suggested Research Connections

- **Scheffer et al. (2001), *Nature*:** Alternative stable states in shallow
  lakes — Restriction + Coupling BC combination; ARW would predict two
  distinct ε-plateaus
- **Holling (1973), resilience theory:** ε-plateau width as resilience
  operationalization — formal connection to ARW stability measures
- **May (1977), thresholds in ecosystems:** Mathematical ecology of tipping
  points — directly maps to regime boundary θ* in ARW
- **Levin (1992), the problem of pattern and scale:** Scale-dependence of
  ecological patterns — this is the ARW scope-dependence problem stated for
  ecology
- **CASE-20260315-0007 ↔ CASE-20260315-0008 transfer:** SIR (Aggregation)
  vs Pitchfork (Symmetry Breaking) — both have analytical θ*, tests whether
  different BC classes with known transitions transfer across domains
