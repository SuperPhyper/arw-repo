# Epsilon-Induced Relational Emergence

## Purpose
To formalize a minimal ARW emergence case where local observables collapse under ε,
while relational observables remain structure-bearing.

## Core Statement
Emergence occurs when Π_local becomes indistinguishable under ε,
while Π_rel remains distinguishable and induces a non-trivial partition.

## Formal Structure
S = (B, Π, Δ, ε)

Π = Π_local ∪ Π_rel

Emergence if:
1. Π_local collapses under ε
2. Π_rel remains distinguishable
3. Π_rel induces non-trivial R_S

## Interpretation
Emergence = selective survival under ε-induced coarse-graining.

## Minimal Example
Coupled oscillators:
- Π_local: amplitudes
- Π_rel: phase difference, synchronization

At suitable ε:
- amplitudes collapse
- synchronization persists

## Implications
- Emergence is scope-relative
- ε acts as observable selector
- relational structure survives local collapse
