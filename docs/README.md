# Documentation

All framework documentation lives here. The layer hierarchy mirrors the
ARW/ART conceptual structure — from formal foundations up to domain applications.

For a fully searchable flat index of every document, see [INDEX.md](INDEX.md).

and [notes/research_journal.md](notes/research_journal.md) for current state.

All contributions must pass the pre-commit check in [meta/LLM_CONTRIBUTION_CHARTER.md](meta/LLM_CONTRIBUTION_CHARTER.md).
