# Data

Storage for data artifacts generated by the pipeline or used in experiments.

---

## Structure

| Directory | Purpose |
|---|---|
| `blobs/` | Binary large objects (simulation outputs, serialized results) |

Case-specific data (sweep results, partition results, invariants) is stored
within the corresponding case directory in [cases/](../cases/).
This directory is reserved for shared or cross-case data artifacts.
