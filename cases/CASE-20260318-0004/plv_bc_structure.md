# PLV Observable BC Structure — CASE-20260318-0004

## Purpose

This document formally characterises the BC structure of the PLV (phase-locking value)
observable used in CASE-20260318-0004 (Coupled Stuart-Landau). It is required for
Control C3 of the pre-registered transfer experiment
(transfer_test_0004_0001_coupling/design/controls_checklist.md) and documents the
observable BC structure contrast with r_ss (CASE-0001) per the ARW/ART principle that
Phi measures observable transfer, not system transfer (established 2026-03-18).

---

## Observable Definition

PLV = |mean(exp(i * (phi_1(t) - phi_2(t))))| over t in [t_transient, T]

where phi_i(t) = arg(z_i(t)) is the instantaneous phase of oscillator i.

- Domain: X_B (joint state space after transient t_transient = 50)
- Codomain: [0, 1]
- PLV = 0: uniform phase distribution (incoherent, independent oscillators)
- PLV = 1: perfect phase locking (fully synchronized)

## Primitive Operator Decomposition

PLV structurally involves the following ARW primitive operators:

### C — Coupling (PRIMARY)

PLV directly and exclusively measures inter-oscillator phase coherence produced by the
diffusive coupling term K(z_j - z_i). This is the Coupling BC operator.

Evidence from sweep data:
- K = 0.01: PLV = 0.158 (near-zero, incoherent — coupling too weak)
- K = 0.06+: PLV = 1.000 (perfect phase locking — coupling dominant)
- Transition at K* = 0.055 is directly produced by the Coupling BC

Without the Coupling BC (K = 0), PLV would be 0 (or near-zero) because Stuart-Landau
oscillators with incommensurate natural frequencies (omega_1 = 1.00, omega_2 = 1.08)
would not phase-lock. The entire informational content of PLV is generated by C.

### A — Aggregation (SECONDARY)

PLV is a temporal mean: mean(exp(i * Delta_phi(t))) over T - t_transient = 150 time steps.
Time-aggregation is intrinsic to the observable definition. Without aggregation,
the instantaneous phase difference exp(i * Delta_phi(t)) is a complex number, not a
real-valued regime-distinguishing observable. The aggregation step (absolute value of mean)
converts it to PLV in [0, 1].

### R — Restriction (TERTIARY)

PLV is computed on the limit cycle attractor. The amplitude restriction (Stuart-Landau
nonlinearity |z_i|^2 term) confines the state space to the attractor manifold |z_i| ~ sqrt(lambda_i).
This Restriction BC shapes which states PLV acts on, but does not generate PLV's informational
content. It is a precondition for the measurement, not the primary driver.

### S — Symmetry (STRUCTURAL PROPERTY)

PLV is invariant under global phase rotation: (phi_1, phi_2) -> (phi_1 + c, phi_2 + c).
This symmetry invariance is built into the phase-difference structure of the observable.
It means PLV cannot distinguish states that differ only by a global phase offset — which
is appropriate for measuring coupling-induced synchronization.

---

## BC Structure Notation

**PLV: C . A . R**

- C: Coupling — primary (generates the observable's informational content)
- A: Aggregation — secondary (time-averaging required for the scalar value)
- R: Restriction — tertiary (attractor confinement as precondition)

---

## Contrast with r_ss (CASE-0001)

r_ss (Kuramoto order parameter) has BC structure **R^3 . A . D** (Restriction-dominated).
This was established in session 2026-03-18.

| Property | PLV (CASE-0004) | r_ss (CASE-0001) |
|----------|----------------|-----------------|
| System BC class | Coupling | Coupling |
| Observable BC structure | C . A . R | R^3 . A . D |
| Primary operator | Coupling (C) | Restriction (R) |
| Observable type | Relational (inter-oscillator) | Collective (mean-field) |
| What it measures | Inter-oscillator phase coherence | Alignment of phases with mean field |
| Collapses at | K = 0 (no coupling) | kappa = 0 (no coupling) |

Both observables collapse when coupling is absent — confirming they are both
Coupling-BC-sensitive. But PLV measures the direct inter-oscillator coupling effect
(relational), while r_ss measures the indirect mean-field restriction effect (collective).

This observable-level BC structure difference did not prevent high Phi = 0.9983 in the
transfer test. The partition structure (N=4, linear adjacency, normalized theta* ~ 0.5)
is conserved across the observable BC mismatch. This is evidence that partition topology
is more structurally invariant than observable BC structure within the same system BC class.

## Implications for Transfer Interpretation

The transfer test result (Phi = 0.9983, highly_admissible) is strengthened by this analysis:

1. The two scopes use observables with different BC structures (C . A . R vs R^3 . A . D).
2. Despite this, both observables recover the same four-regime partition with linear
   adjacency and nearly identical normalized transition positions (TBS = 0.008).
3. This confirms the hierarchical factorisation hypothesis: same BC class at the system
   level is the dominant factor in transfer admissibility, and the observable BC structure
   difference is a second-order effect that does not break partition compatibility.

This finding is relevant for open question Q-EMERGE-03 (CaseRecord.yaml): relational
observables (PLV) appear to be at least as transferable as collective observables (r_ss)
across same-BC-class Coupling systems.
